# Desarrollo web con Rust

Rust ofrece seguridad de memoria, rendimiento cercano a C y un ecosistema web maduro.

## Frameworks destacados

- **Axum** — ergonómico, basado en Tower y Tokio
- **Actix-web** — alto rendimiento, actor model
- **Leptos** — fullstack con soporte WASM

## Conclusión

Con herramientas como `leptos-content-collection` puedes tener un blog
completamente tipado sin base de datos ni CMS externo.
